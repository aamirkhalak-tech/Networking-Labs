# EIGRP and Redistribution Lab

## Objective:
To configure EIGRP routing and perform route redistribution between networks.

## Network Topology
- 4 routers connected in sequence
- Each router connected to a LAN via a switch
- Multiple PCs placed in different networks
- End-to-end communication established between all networks

## What I Configured:
- Enabled EIGRP routing
- Configured network statements
- Performed redistribution between routing protocols

## Commands Used:
show ip route
show ip protocols

## Outcome:
Achieved successful communication between different routing domains using redistribution.
